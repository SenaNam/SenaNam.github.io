10년 가까운 웹 디자인 경력을 바탕으로, 단순히 "보기 좋은" 화면을 넘어 기술적으로 완벽하게 구현되는 프론트엔드 개발자가 되었습니다.

UI/UX 설계부터 HTML, CSS, JavaScript 구현까지—디자인과 개발의 경계를 자유롭게 넘나들며 시각적 완성도와 기술적 신뢰성을 동시에 갖춘 결과물을 만들어냅니다.

Java, Spring Boot 기반 백엔드 구조를 이해하고 REST API 연동까지 직접 수행할 수 있는 풀스택 역량도 보유하고 있습니다.
